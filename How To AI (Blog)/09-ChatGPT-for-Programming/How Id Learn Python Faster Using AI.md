## The AI-Assisted Learning Stack
- **ChatGPT or Claude** as your tutor
- **GitHub Copilot** (free) in VS Code as your coding assistant
- No paid courses needed

## 30-Day Approach
1. Install Python + VS Code + Copilot
2. Ask AI to walk you through basics step by step
3. Write comments first, let Copilot suggest code
4. Read before accepting — type it yourself first
5. Build a real project by day 30

## Key Prompt
```
I'm building [project]. I need to [task].
Here's what I've tried: [paste code].
What's wrong and how do I fix it?
Explain why, don't just fix it.
```

The "explain why" keeps you learning instead of copying.
