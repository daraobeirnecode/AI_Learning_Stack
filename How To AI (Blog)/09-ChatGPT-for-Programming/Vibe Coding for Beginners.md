## What Is Vibe Coding?
Describing what you want in English and accepting whatever the AI codes. You don't read the code — you test the result.

## The Process
1. Describe your goal in plain English
2. Let Claude Code (or similar) write everything
3. Check the output (click every button, test on phone)
4. Note issues → tell AI → repeat

## Tools
- **Claude Code** — best agentic coding tool (see [[Claude Code for Non-Coders Part 1 - Setup and Basics]])
- **Lovable / Replit / Bolt.new** — app builders (describe → get working app)
- **Cursor / Windsurf** — AI-enhanced code editors
