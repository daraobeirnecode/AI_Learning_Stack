## Ruben's 8 Essential AI Tools

| Tool | Purpose | Why It Wins |
|------|---------|-------------|
| **Claude** (Cowork) | Writing, thinking, file work | Gets it right first try with .md context |
| **Grok** | Real-time search | Connected to X — fastest news |
| **Wispr Flow** | Voice-to-text | Talk instead of type; 200+ WPM |
| **Granola** | Meeting notes | Silent note-taking (no bot joining) |
| **Gamma** | Slides/presentations | One prompt → full deck |
| **NanoBanana Pro** | AI images | Clean, believable, editable |
| **NotebookLM** | Source-based research | Upload sources, ask anything |
| **OpusClip** | Video repurposing | One link → 20–40 shorts |

## The Big Picture
- **Claude** → best for writing, coding, file-based work
- **ChatGPT** → jack-of-all-trades, images, video, GPTs, web agent
- **Gemini** → best free tier, best images (Imagen 4), best video (Veo-3)
- **Grok** → best for real-time news via X connection
- **Perplexity** → best for cited research
