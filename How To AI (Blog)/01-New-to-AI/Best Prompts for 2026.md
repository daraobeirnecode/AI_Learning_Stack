## Core Insight
In 2026, **context > prompts**. The best "prompt" is an .md file with your identity and rules.

## Universal Template
```
I want to [TASK] so that [SUCCESS LOOKS LIKE].
Read the uploaded files first.
DO NOT start executing. Ask me clarifying questions first.
Only begin once we've aligned.
```

## 4W Context Prompt (ChatGPT)
```
ROLE: Act as a [expert role].
Voice/tone: [style]. Length: [target].
Key facts to use: [numbered list].
Constraints: [boundaries].
Steps: 1. Gap check 2. Plan 3. Draft 4. Review 5. Revise
```
