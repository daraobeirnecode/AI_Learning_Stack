## Context Engineering (Beyond Prompting)
The right context beats the right prompt. Ruben's 4W framework:
- **Who** — Role assignment with specific expertise
- **What** — Task with success criteria
- **With** — Context package (files, data, constraints)
- **Way** — Step-by-step workflow with checkpoints

## Reducing Hallucinations
- "ONLY use the sources I uploaded"
- "Answer using this template: According to [source]..."
- Upload specific files rather than asking for general knowledge
