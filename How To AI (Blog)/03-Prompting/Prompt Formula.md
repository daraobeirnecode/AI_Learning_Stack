# Prompt Formula

> **Source:** Ruben Hassid, *How to AI*
> **Section:** [[How to AI - Index#📂 Prompting]]

---

## The Formula
```
ROLE: Act as a [specific expert].
Voice/tone: [style description].
Length: [target].
Key facts/data to use: [numbered list].
Constraints: [boundaries].

Steps:
1. Gap check — list missing info; ask questions
2. Plan — outline structure; wait for approval
3. Draft — write following the plan
4. Review — ask for feedback on clarity, tone, completeness
5. Revise — improve with notes; repeat until AGREE
```

## Key Principles
- Context is more important than the prompt itself
- Upload files > writing long prompts
- Force AI to ask YOU questions before executing
- Use "Think deeply" at the end of every prompt (for reasoning models)
- One deliverable per prompt
