# ChatGPT Overview — GPT-5, Features, Plans

> **Source:** Ruben Hassid, *How to AI*
> **Section:** [[How to AI - Index#📂 ChatGPT]]

---

## Current Capabilities (2026)
- Browse the web, create 20-page reports with Deep Research
- Store memories, remember past conversations
- Generate videos with Sora, make images with DALL-E
- Agent mode for web automation
- Projects for organized context
- Voice Mode for spoken interaction
- Custom GPTs for niche purposes

## Plans
- Free tier — solid for casual use
- Plus — more messages, model access
- Pro — Deep Research, advanced features
