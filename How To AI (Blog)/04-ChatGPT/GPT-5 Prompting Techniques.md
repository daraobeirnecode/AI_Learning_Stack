## Key Techniques for GPT-5
1. Always add "Think deeply" to trigger reasoning
2. Use the 4W context prompt framework (see [[Prompt Formula]])
3. Upload files for context instead of describing them
4. Use Deep Research mode for multi-source investigations
5. Challenge outputs — "Can we review this?" / "Search online for comparable pricing"
