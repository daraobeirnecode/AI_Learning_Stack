# How to Go from ChatGPT Beginner to Pro

> **Source:** Ruben Hassid, *How to AI*
> **Section:** [[How to AI - Index#📂 ChatGPT]]

---

## Key Steps
1. Stop using default ChatGPT — customize Personalization settings
2. Choose the right model (GPT-5 Thinking for important tasks)
3. Always add "Think deeply" to prompts
4. Use Deep Research for multi-source reports
5. Use Projects to separate work contexts
6. Try Agent mode for web-based tasks
