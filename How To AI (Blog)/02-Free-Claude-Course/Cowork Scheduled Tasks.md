# Cowork Scheduled Tasks

> **Source:** Ruben Hassid, *How to AI*
> **Section:** [[How to AI - Index#📂 Free Claude Course (Beginner to Pro)]]

---

## What It Is
Set up recurring tasks in Cowork that run automatically — daily, weekly, or on a schedule. Combined with Cowork Projects (persistent memory), scheduled tasks remember context between runs.

## Examples
- Weekly report generation from project folders
- Daily email digest summarization
- Recurring client update drafts
- Automated data analysis on updated spreadsheets

## Setup
Inside a Cowork Project, define a task and set its schedule. Claude executes it in the background and delivers results to your CLAUDE OUTPUTS folder.
