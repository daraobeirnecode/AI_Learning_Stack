## What Cowork Does
Claude Cowork is the "Claude Code of knowledge workers." It reads your files, creates documents, builds spreadsheets, writes code you'll never see, and asks clarifying questions.

## Real Workflow Examples

### Newsletter Writing
Drop voice profile + anti-AI guide in folder. Claude reads everything, generates a form asking about audience/tone/length, produces an outline, you push back, it adjusts, then writes — sounding like you.

### Client Deliverable
Drop brief + templates + past deliverables in folder:
```
A client just sent a brief for a 2026 AI adoption strategy.
Read the brief, my deliverable template, and my past examples.
Create a first draft as a .docx.
```

### Excel from Scratch
```
Create a professional Excel spreadsheet (.xlsx) for: [PURPOSE]
Context: [WHO IS IT FOR]
Rules:
- Use Excel formulas — never hardcoded calculations
- Put editable assumptions in labeled cells
- Freeze top row, auto-fit columns
```

### Plugin Workflows
- `/marketing:draft-content` → LinkedIn post using your voice profile
- `/data:explore` → Drop CSV, get summaries + anomalies + dashboard
- `/legal:review` → Flag risky clauses in a contract
