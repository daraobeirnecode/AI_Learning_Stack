## Essential Setup Checklist
1. Download desktop app: [claude.com/download](https://claude.com/download)
2. Subscribe to Pro ($20/month) or Max ($100/month)
3. Set default model: **Opus 4.6**
4. Enable **Extended Thinking** (always)
5. Set up Connectors (Settings → Connectors): Gmail, Google Drive, Slack, Notion
6. Create your Cowork folder structure (see [[Cowork Setup]])
7. Write your about-me.md (see [[How to Make Claude Write Like You]])
8. Set Global Instructions (see [[Cowork Setup#Step 3 Set Global Instructions]])

## Model Selection Guide
| Model | Use Case |
|-------|----------|
| Opus 4.6 | Everything important — writing, thinking, analyzing |
| Sonnet 4.5 | Quick tasks to conserve usage |
| Haiku 4.5 | Lightest tasks |
