-e ## Core Concepts
- **Vibecoding**: Describing what you want in English and letting Claude write all code
- **Auto accept edits**: Skip permission prompts (use with VS Code extension)
- **The feedback loop**: Check site → note issues → tell Claude → repeat
- **One deliverable per prompt**: Don't dump 12 features in one request
- **Screenshot debugging**: Paste a screenshot of a bug into Claude Code, say "fix this"
