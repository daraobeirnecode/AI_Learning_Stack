## Core Insight
In 2026, Claude is not one tool — it's six:

1. **Cowork** — desktop app that works on your actual files (most important)
2. **Model** — use Opus 4.6 + Extended Thinking (most people use the wrong model)
3. **Excel** — AI inside your spreadsheets that reads/writes formulas
4. **Plugins** — pre-built skill packs (Sales, Marketing, Legal, Finance, Data, etc.)
5. **Artifacts** — interactive outputs (calculators, charts, trackers)
6. **Projects** — persistent context folders (Ruben no longer uses these, prefers Cowork)

## Why Claude Over ChatGPT
- Claude Cowork reads your files, creates documents, builds spreadsheets, writes code
- It asks *you* clarifying questions instead of guessing wrong
- With markdown files providing context, you barely need to prompt
- 1M+ token context window means minimal hallucination

## Setup Steps
1. Download at [claude.com/download](https://claude.com/download)
2. Get Pro ($20/month) or $17/month annually
3. Open app → click **Cowork** tab
4. Select a folder from your computer
5. **Always select Opus 4.6 + Extended Thinking**

## The Mindset Shift
ChatGPT trained you to write better prompts. With Claude Cowork, the game is **text files**.

Take everything you know (writing style, brand rules, best examples, past work) → put in .md files → drop in a folder → point Claude to that folder.

**One great markdown file > 50 random uploads.**

## First Prompt Template
```
I want to [YOUR TASK] so that [WHAT SUCCESS LOOKS LIKE].

First, read the uploaded files completely before responding.

DO NOT start executing yet. Instead, ask me clarifying questions 
(use AskUserQuestion) so we can refine the approach together step by step.

Only begin work once we've aligned.
```

## Connectors (Internet Access)
Settings → Connectors → Browse directory → Add (Slack, Google Drive, Notion, Figma, 50+ others). Free on all plans.

## Where Claude Falls Short
- **No image generation** — use Gemini + Nano-Banana 2
- **Not the best at real-time search** — use Grok (connected to X)
- **No video generation** — use Seedance 2.0 or Gemini VEO-4
