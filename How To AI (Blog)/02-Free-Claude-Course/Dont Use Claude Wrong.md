# Don't Use Claude Wrong

> **Source:** Ruben Hassid, *How to AI*
> **Section:** [[How to AI - Index#📂 Free Claude Course (Beginner to Pro)]]

---

## Common Mistakes
1. **Using Chat instead of Cowork** — Chat is only 20% of Claude's value
2. **Using the wrong model** — always Opus 4.6 + Extended Thinking
3. **Writing long prompts instead of creating .md files** — context > prompts
4. **Not using AskUserQuestion** — let Claude ask YOU questions first
5. **Dumping hundreds of files** — be intentional about quality over quantity
6. **Expecting perfection on first try** — iterate, push back, give feedback
7. **Not setting up Global Instructions** — set once, runs every time
8. **Using Projects instead of Cowork** — Projects are chat with better memory; Cowork actually creates files and runs code

## The Right Way
- Start every prompt with: "Read the files, then ask me questions"
- Use Cowork with a structured folder
- Create an about-me.md and anti-ai-writing-style.md
- Push back on bad output: "This sounds like AI. Rewrite [part]."
