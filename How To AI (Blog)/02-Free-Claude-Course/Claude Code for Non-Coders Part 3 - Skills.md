-e ## Claude Code Skills
Skills are reusable markdown instructions Claude automatically applies. They live in your Cowork folder and activate when a task matches.

### Creating Skills
1. Open Cowork → type: "Use the skill-creator to help me build a skill for [task]"
2. Answer the interview questions (like the taste interview)
3. Claude generates a SKILL.md file with trigger command and instructions

### Key Facts
- Claude only reads the 3-line header of each Skill initially
- Full instructions load only when a task matches
- 30 installed Skills barely touch your context window
- A 15-message task without a Skill → 2 messages with one

### Troubleshooting
- If a Skill fires when you don't want it, add negative boundaries
- If Claude cuts corners, add to your prompt: "Take your time. Quality over speed."
