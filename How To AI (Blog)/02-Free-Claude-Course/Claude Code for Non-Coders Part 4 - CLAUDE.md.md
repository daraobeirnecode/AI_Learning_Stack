-e ## What Is CLAUDE.md?
A markdown file in your project root that Claude Code reads first every session. It's your project's persistent memory.

### What to Include
- Font choices, color palette, page structure
- Tech stack and architecture decisions
- Past edits and current state
- Style rules and conventions

### Why It Matters
Without CLAUDE.md, every Code session starts from zero. With it, you say "Add a blog section" and Claude builds it in the exact same style as everything else.

### Developers have been doing this for months. Now you can too.
