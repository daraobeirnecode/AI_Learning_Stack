## What Is an .md File?
A markdown file — plain text with an .md extension. Claude reads them better than .txt files. Open any text editor, write content, save as `about-me.md`.

## The Two Essential Files

### 1. about-me.md
Contains:
- What you do for work
- How you communicate (formal? casual? direct?)
- One example of writing you're proud of
- Current priorities and projects

### 2. anti-ai-writing-style.md
Contains your "HARD NOs" — everything that makes writing sound like AI:
- Banned phrases, banned words, banned structures
- Specific do's and don'ts for tone
- Examples of good vs bad output

## How to Create Yours
Use the 100-question taste interview from [[How to Make Claude Write Like You]]. Or start simpler — write 3 things about who you are and how you work.

**Pro tip:** Use voice-to-text (like Wispr Flow) instead of typing.

## Where to Put Them
In your Cowork folder under `ABOUT ME/` — see [[Cowork Setup]] for full structure.
