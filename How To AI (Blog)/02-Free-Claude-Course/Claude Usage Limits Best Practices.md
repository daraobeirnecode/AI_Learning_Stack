# Claude Usage Limits — Best Practices

> **Source:** Ruben Hassid, *How to AI*
> **Section:** [[How to AI - Index#📂 Free Claude Course (Beginner to Pro)]]

---

## Plans & Limits
| Plan | Price | Approx Messages/5hr |
|------|-------|---------------------|
| Free | $0 | Limited |
| Pro | $20/mo | ~45 messages |
| Max | $100–200/mo | Much higher |

## What Burns Usage Fast
- Claude Code sessions (1 session ≈ 20+ regular chats)
- Cowork with large file reads
- Extended Thinking on complex tasks
- Long context conversations

## How to Conserve Usage
- Use Sonnet 4.5 or Haiku 4.5 for simple tasks
- Use .md files + Global Instructions so prompts stay short
- Skills load only their 3-line header until triggered (30 Skills barely touch context)
- A task taking 15 back-and-forth messages without a Skill → 2 messages with one

## When to Upgrade to Max
If you use Cowork or Code daily, Pro will feel tight within a week. Max is worth it for power users.
