## The Problem Cowork Had
Cowork forgot everything. Every session started from zero. You had to re-explain context and re-upload files repeatedly.

## The Fix: Projects Inside Cowork
Anthropic merged Projects into Cowork. Each Project is now a persistent workspace with:
- **Scoped memory** — say "build on last week's report" and Claude knows what it was
- **Custom instructions** baked in per project
- **Dedicated folders**
- **Scheduled tasks**

## Quick Setup (12 Minutes)
**Minutes 0–1:** Update Claude Desktop app (Projects needs March 2026 update).

**Minutes 1–8:** Create first Cowork Project:
- Open Cowork → Projects → + → Pick method (scratch, import, or existing folder)
- Write instructions (5–8 lines max)

**Minutes 8–12:** Run first task:
```
Read everything in this folder. Then [your task].
Start with AskUserQuestion so we align before you execute.
```
